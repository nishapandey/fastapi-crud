FastAPI CRUD Project

Description:
This project demonstrates a simple CRUD (Create, Read, Update, Delete) application using FastAPI.

Getting Started:
1. Install dependencies:
   pip install fastapi uvicorn

2. Run the application:
   python main.py

3. Access the API:
   Visit http://localhost:8000 in your browser or use an API client.

Files:
- main.py: Entry point for the application.
